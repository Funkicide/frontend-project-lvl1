# Brain games app
App consists of five logic based console games: Brain Even, Brain Calc, Brain Gcd, Brain Progression, Brain Prime.

## System requirements

* NodeJS >= 12 <= 18
* NPM >= 6.x
* Make

## Setup

```sh
make install
```

## Using

```sh
make brain-even
make brain-calc
make brain-gcd
make brain-progression
make brain-prime
```
